# Taskify – Full Stack Task Management App

Taskify is a full-stack web application to manage tasks with authentication. Built with:

- 🧑‍🎨 React (Vite) – Frontend
- 🚀 Node.js + Express – Backend
- 🧠 MongoDB – Database
- 🔐 JWT Authentication

---

## 🚀 Features

- User login system
- Add and delete tasks
- Task priority levels (High, Medium, Low)
- JWT-secured API routes
